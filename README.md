# Simple Social Media Application

This is a basic social media application built using Streamlit in Python. The app allows users to:

- Register and log in with an email and password
- Create posts by uploading images or videos along with captions
- View a home feed of all the posts shared by different users
- Delete their own posts

## Features

- User authentication (register, login)
- Post creation (images/videos with captions)
- Feed display with user information and delete option for post owners

This project is intended to demonstrate basic web application functionality using FastAPI and Streamlit.

## Requirements

- Python 3.x
- Streamlit
- Requests library
